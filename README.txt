Author: Maverick Peppers
Description: Project folder for Smartvu problem solutions
Date: 6/7/2016

To use, open your favorite CLI and set your working path to the project directory
run `npm install --save` to download node module dependancies
run `node script.js -h` to see the script help manual
Sample output below.

#####################
node script.js --help

  Usage: main [options] <problem>

  Options:

    -h, --help                                output usage information
    -o, --output <output filename>            save program output as a text file
    -a, --array <comma seperated array list>  e.g.: 5,6,23,5,90,.... If no array list is provided, a default one will be used.
#####################
